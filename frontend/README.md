## 📖 專案簡介

逃兵大陸前端專案，使用 Vue 3 + Vite 開發，提供互動式的兵役資訊與閃兵傳奇故事。

## ✨ 主要功能

- 🏠 **首頁** - 功能導覽與網站簡介
- 🎖️ **國軍簡介** - 台灣三軍介紹
- 📋 **新兵指南** - 入伍準備與軍中須知
- 🏃 **閃兵傳奇** - 藝人閃兵故事展示
- 🎲 **抽籤體驗** - 互動式抽籤模擬
- 🔥 **Firebase 整合** - 使用者認證與資料存取

## 🛠️ 技術棧

- **Vue 3** - Composition API
- **Vite** - 極速建置工具
- **Vue Router 4** - 路由管理
- **Pinia** - 狀態管理
- **Tailwind CSS 4** - CSS 框架
- **GSAP** - 動畫效果
- **Font Awesome** - 圖標庫
- **SweetAlert2** - 互動式訊息
- **Firebase** - 認證與資料庫

## 📱 頁面路由

- `/` - 首頁
- `/military` - 國軍簡介
- `/new-soldier-guide` - 新兵指南
- `/deserters` - 閃兵傳奇列表
- `/deserters/:id` - 閃兵詳情
- `/lottery` - 抽籤體驗
- `/about` - 關於頁面

## 專案簡介

「逃兵大陸」是一個以輕鬆幽默的方式介紹台灣軍事文化和兵役制度的網站。本網站包含四個主要頁面，提供國軍簡介、新兵指南、閃兵傳奇等豐富內容，適合即將入伍或對兵役好奇的民眾瀏覽。

### 🎯 專案目標
- 以幽默方式降低新兵對入伍的焦慮
- 提供實用的軍中生存指南
- 介紹台灣國軍的歷史與現況
- 展示完成逃兵役的知名藝人

## 檔案結構

```
Deserter/
├── 📄 index.html                 # 首頁
├── 📄 introduction.html          # 國軍簡介頁
├── 📄 recruits.html             # 新兵入伍指南頁
├── 📄 deserter.html             # 閃兵傳奇頁
├── 📄 README.md                 # 專案說明文件
│
├── 📁 css/                      # 樣式與腳本目錄
│   ├── 🎨 styles.css           # 全域樣式
│   ├── 🎨 introduction.css     # 國軍簡介頁樣式
│   ├── 📜 introduction.js      # 國軍簡介頁腳本
│   ├── 🎨 recruits.css         # 新兵指南頁樣式
│   ├── 📜 recruits.js          # 新兵指南頁腳本
│   ├── 🎨 deserter.css         # 閃兵傳奇頁樣式
```
